---
title: 在 Windows Server 2012 (R2) 上显示 “计算机”
date: 2015-03-18
tags: ['技术']
---

Windows Server 2012 (R2)的桌面上，默认是没有“计算机”的，而且桌面上右键也是不会出现个性化的，如下方法可以迅速打开“桌面图标设置”：

1、在左下角 Windows 徽标上右键，点击“运行”

2、在“运行”中输入如下代码：

``` bash
rundll32.exe shell32.dll,Control_RunDLL desk.cpl,,0
```

3、将“计算机”的图标勾上

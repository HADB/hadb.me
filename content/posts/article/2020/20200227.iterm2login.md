iterm2login.sh 文件：

```bash
#!/usr/bin/expect

set timeout 30
spawn ssh -p [lindex $argv 0] [lindex $argv 1]@[lindex $argv 2]
expect {
        "(yes/no)?"
        {send "yes\n";exp_continue}
        "password:"
        {send "[lindex $argv 3]\n"}
}
interact
```

复制配置到其他 Profile：Other Actions -> Bulk Copy from Selected Profile

Session -> Status Bar

iTerm2 -> Install Shell Integration
